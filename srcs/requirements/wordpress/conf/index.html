Inception : what are docker containers ?

Docker containers are lightweight and standalone executable packages to run an applications. This makes them particularly portable.

They share the host's operating system's kernel, making them more lightweight and ressource efficient than Virtual Machines. They provide process-level isolation, isolating the application and it's dependencies. But this isolation is less strict than the one of Virtual Machines.
